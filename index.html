Get Started

  1. Open your favorite web editor. Most will do (Sublime, Atom, VS Code, BBedit, Brackets). I'm using Sublime. Make a folder to save your files.
  2. Grab the CDN link to p5.min.js and add it in a script tag at the bottom of your page.
  3. Check out the p5 reference
